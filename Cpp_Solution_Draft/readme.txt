

Для сборки проекта можно использовать коробочный дистрибутив Linux стандартной комплектации
(например, Ubuntu 20.04 и позже):

NB! Для сборки под Ubuntu 20.04 достаточно пакетов с инсталяционного диска (без необходимости загрузки
    пакетов с удаленных внешних репозиториев)

Требуется компилятор с поддержкой C++11 и стандартным набором библиотек (например, g++ или clang++)

Пример сборки проекта:

g++ -std=c++11 nuke.cpp -o nuke

Команда запуска проекта:

./nuke <файл_с_координатами_целей> <радиус_действия_боеприпаса>

Примеры запуска (с выводом результата работы программы):

./nuke coords.txt 10

       3 24 45

./nuke coords5.txt 10

       47.999999999999996447286321199499070644378662109375 25.999999999999989341858963598497211933135986328125 2
