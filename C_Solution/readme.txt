

Для сборки проекта можно использовать коробочный дистрибутив Linux стандартной комплектации
(например, Ubuntu 20.04 и позже):

NB! Для сборки под Ubuntu 20.04 достаточно пакетов с инсталяционного диска (без необходимости загрузки
    пакетов с удаленных внешних репозиториев)

Требуется стандартный компилятор C и стандартный набор библиотек (например, gсс или clang)
Возможна сборка программы для разных архитектур процессора

Пример сборки проекта и запуска:

* Для архитектуры X64
    gcc -DARCH_X64 nuke.c -o nuke
    ./nuke coords.txt 10         
    29.692076421335645619592469302006065845489501953125 75.012190121022086941593443043529987335205078125 5

* Для архитектуры X32
    gcc -DARCH_X32 nuke.c -o nuke
    ./nuke coords.txt 10         
    29.69207763671875 75.012176513671875 5

* Для архитектуры X16
    gcc -DARCH_X16 nuke.c -o nuke
    ./nuke coords.txt 10         
    30 75 5
